About

A Library Management System helps librarians and users efficiently manage books, records, and transactions. It digitizes the process of tracking available books, issued books, and returned ones.

Features

  • Book cataloging and search
  • Issue and return management
  • User membership system
  • Fine calculation for late returns
  • Reports and statistics
